## はじめに

前回は：

* 2量子ビット状態

* ベル状態
* エンタングルメント

を扱いました。

ここで：

量子ビット同士が結びつく状態

を作れるようになりました。

今回は：

**Deutsch–Jozsaアルゴリズム**

を実装します。

これは：

量子アルゴリズムとして最初に学ぶべき代表例

そして：

量子並列性とは何か？

を初めてコードとして体験できます。

---

# 問題設定

Deutsch–Jozsaアルゴリズムは：

ある関数 f(x)

が

定数関数か
バランス関数か

を判定する問題です。

---

# 定数関数とは何か？

定数関数とは：

すべての入力に対して

同じ値を返す関数です。

例：

f(0) = 0
f(1) = 0

または：

f(0) = 1
f(1) = 1

---

# バランス関数とは何か？

バランス関数とは：

半分が0
半分が1

になる関数です。

例：

f(0) = 0
f(1) = 1

または：

f(0) = 1
f(1) = 0

---

# 古典コンピュータなら何回必要か？

古典コンピュータでは：

最低2回

関数を評価する必要があります。

つまり：

f(0)
f(1)

を両方計算する必要があります。

---

# 量子コンピュータなら何回必要か？

量子コンピュータでは：

1回

で判定できます。

これが：

量子並列性

の最初の例です。

---

# アルゴリズムの全体構造

Deutsch–Jozsaアルゴリズムは：

次の流れで動きます。

1. 初期状態を準備
2. 重ね合わせを作る
3. オラクルを適用
4. 干渉を起こす
5. 測定する

順番に実装していきます。

---

# 回路の準備

まず：

量子回路を作ります。

コード：

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(2, 1)
```

ここでは：

入力ビット：1
補助ビット：1

を使用します。

---

# 初期状態を準備する

補助ビットを：

|1⟩

にします。

```python
qc.x(1)
```

次に：

Hadamardを適用します。

```python
qc.h(0)
qc.h(1)
```

これで：

重ね合わせ状態

が作られます。

---

# なぜ補助ビットが必要なのか？

Deutsch–Jozsaでは：

関数を

次の形で表現します：

|x⟩|y⟩ → |x⟩|y ⊕ f(x)⟩

つまり：

関数の値を

補助ビットに書き込みます。

---

# オラクルとは何か？

量子アルゴリズムでは：

関数そのものを

オラクル

として表現します。

つまり：

ブラックボックス関数

です。

今回は：

バランス関数を実装します。

例：

f(x) = x

---

# バランス関数オラクルを作る

f(x)=x の場合：

CNOTで表現できます。

```python
qc.cx(0, 1)
```

これが：

オラクルです。

---

# 干渉を起こす

次に：

もう一度Hadamardを適用します。

```python
qc.h(0)
```

ここで：

干渉

が起きます。

---

# 測定する

最後に：

測定を追加します。

```python
qc.measure(0, 0)
```

これで：

結果が得られます。

---

# 全体コード

まとめると：

```python
from qiskit import QuantumCircuit
from qiskit_aer import AerSimulator
from qiskit.compiler import transpile

qc = QuantumCircuit(2, 1)

qc.x(1)

qc.h(0)
qc.h(1)

qc.cx(0, 1)

qc.h(0)

qc.measure(0, 0)

simulator = AerSimulator()

compiled = transpile(qc, simulator)

job = simulator.run(compiled, shots=1000)

result = job.result()

print(result.get_counts())
```

---

# 実行結果

結果：

{'1': 1000}

つまり：

必ず1

になります。

これは：

バランス関数

であることを意味します。

---

# 定数関数の場合

定数関数の場合：

オラクルを削除します。

つまり：

CNOTを削除します。

```python
qc = QuantumCircuit(2, 1)

qc.x(1)

qc.h(0)
qc.h(1)

qc.h(0)

qc.measure(0, 0)
```

結果：

{'0': 1000}

になります。

---

# なぜ1回で判定できるのか？

量子コンピュータでは：

|0⟩
|1⟩

を同時に計算できます。

つまり：

f(0) と f(1)

を同時に評価しています。

これが：

量子並列性

です。

---

# 重要なのは並列計算ではない

ただし：

ここで重要な注意があります。

量子計算は：

単なる並列計算

ではありません。

重要なのは：

干渉

です。

干渉によって：

正しい答えだけが残ります。

---

このアルゴリズムは：

* 干渉

これらすべてが：

量子アルゴリズムの基本構造

です。
