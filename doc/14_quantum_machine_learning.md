# 量子機械学習とは何か？

量子機械学習とは：

量子回路を使って

機械学習問題を解く

アプローチです。

目的：

* 学習の高速化
* 高次元特徴空間の活用
* 新しいモデル構造の探索

です。

---

# なぜ量子機械学習が注目されているのか？

理由は3つあります：

① 状態空間が指数的に広い
② 干渉を利用できる
③ 重ね合わせで並列探索できる

つまり：

従来とは異なる特徴表現

が可能になります。

---

# 古典機械学習との違い

古典機械学習：

ベクトルを操作する

例：

```
x = (x1, x2, x3)
```

量子機械学習：

量子状態を操作する

```
|ψ⟩
```

つまり：

データ表現の次元が変わります。

---

# 量子機械学習の基本構造

基本構造は：

ハイブリッド計算

です。

流れ：

1 データを量子状態に変換
2 量子回路を適用
3 測定する
4 古典最適化する
5 繰り返す

これは：

VQEやQAOAと同じ構造です。

---

# データの量子化とは何か？

機械学習では：

データを回路に入力する必要があります。

例：

```
x → θ
```

として：

回転ゲートに変換します。

例：

```python
qc.ry(x, 0)
```

これを：

**特徴マップ（feature map）**

と呼びます。

---

# 特徴マップとは何か？

特徴マップとは：

古典データを

量子状態に埋め込む方法

です。

例：

```
x → |ψ(x)⟩
```

この変換によって：

量子空間で分類可能になります。

---

# なぜ特徴マップが重要なのか？

機械学習では：

特徴設計

が最も重要です。

量子機械学習では：

特徴マップ

がそれに対応します。

つまり：

性能を決める中心要素です。

---

# 量子回路はニューラルネットになる

変分量子回路は：

ニューラルネットと似ています。

比較：

ニューラルネット：

```
y = f(Wx + b)
```

量子回路：

```
|ψ⟩ → U(θ)|ψ⟩
```

ここで：

θ

が学習対象になります。

---

# 量子ニューラルネットとは何か？

量子ニューラルネット（QNN）は：

パラメータ付き量子回路

です。

例：

```python
qc.ry(theta, 0)
qc.rx(phi, 0)
```

これを：

訓練します。

つまり：

回路そのものを学習します。

---

# 最も単純な量子分類器を作る

簡単な分類器を作ってみます。

例：

```python
from qiskit import QuantumCircuit
import numpy as np

def classifier(x, theta):
    qc = QuantumCircuit(1)
    qc.ry(x, 0)
    qc.rz(theta, 0)
    return qc
```

ここで：

x = 入力データ
theta = 学習パラメータ

です。

---

# 測定が出力になる

量子分類器では：

測定結果が出力になります。

例：

```
P(0) > 0.5 → クラスA
P(1) > 0.5 → クラスB
```

つまり：

確率が予測値です。


```
⟨ψ(x)|ψ(y)⟩
```

つまり：

状態の重なり

を測定します。
