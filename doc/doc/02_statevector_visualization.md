# 状態ベクトルとは何か？

量子ビットは：

```

|ψ⟩ = a|0⟩ + b|1⟩

```

という形で表されます。

ここで：

| 記号 | 意味 |
|------|------|
| a | |0⟩になる強さ（確率振幅） |
| b | |1⟩になる強さ（確率振幅） |

そして：

```

|a|² + |b|² = 1

```

になります。

つまり：

確率は

```

係数の2乗

```

で決まります。

# 状態ベクトルを取得する

まず新しいファイルを作ります：

```

statevector_demo.py

````

コードはこちら：

```python
from qiskit import QuantumCircuit
from qiskit.quantum_info import Statevector

qc = QuantumCircuit(1)

qc.h(0)

state = Statevector.from_instruction(qc)

print(state)
````

実行：

```
python statevector_demo.py
```

結果：

```
Statevector([0.70710678+0.j, 0.70710678+0.j],
            dims=(2,))
```

---

# 出力の意味を理解する

この結果：

```
[0.707, 0.707]
```

は：

```
|ψ⟩ = 0.707|0⟩ + 0.707|1⟩
```

を意味します。

つまり：

| 状態 | 係数 | 確率    |     |
| -- | -- | ----- | --- |
|    | 0⟩ | 0.707 | 0.5 |
|    | 1⟩ | 0.707 | 0.5 |

となります。

これが：

Hadamardゲートの正体です ✨

# 別の初期状態を試してみる

from qiskit import QuantumCircuit
from qiskit.quantum_info import Statevector

# なぜ符号が重要なのか？

量子計算では：

```
確率だけでなく
位相が重要
```

です。

つまり：

```
+0.707
−0.707
```

の違いが：

干渉を生みます。

この干渉によって：

* Groverが速くなる
* QFTが機能する
* Shorが動く

ようになります。

量子アルゴリズムの核心は：

```
位相操作
```

です。

---

# 状態をベクトルとして理解する

量子状態は：

ベクトルです。

例えば：

```
|0⟩ = [1, 0]
|1⟩ = [0, 1]
```

Hadamard後は：

```
[0.707, 0.707]
```

つまり：

```
45度方向のベクトル
```

になります。

量子計算とは：

```
ベクトルを回転させる操作
```

なのです。

---

# Bloch球で可視化する

量子状態は：

Bloch球

で表現できます 🌍

追加ライブラリ：

```python
from qiskit.visualization import plot_bloch_multivector
```

コード：

```python
from qiskit import QuantumCircuit
from qiskit.quantum_info import Statevector
from qiskit.visualization import plot_bloch_multivector
import matplotlib.pyplot as plt

qc = QuantumCircuit(1)

qc.h(0)

state = Statevector.from_instruction(qc)

plot_bloch_multivector(state)

plt.show()
```

実行すると：

Bloch球の赤道上にベクトルが表示されます。

これは：

```
重ね合わせ状態
```

を意味します。

# 測定すると何が起きるのか？

ここで重要なポイントがあります。

状態：

```
0.707|0⟩ + 0.707|1⟩
```

を測定すると：

```
どちらか一方
```

だけになります。

つまり：

```
測定 = 状態の破壊
```

です。

量子計算では：

```
測定前にすべてを終わらせる
```

必要があります。

量子アルゴリズムを書くとき：

例えば：

* oracleが正しいか？
* 干渉が起きているか？
* 位相が反転しているか？

は：

状態ベクトルを見ると分かります。
