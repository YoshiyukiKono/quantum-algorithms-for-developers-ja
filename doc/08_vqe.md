---

# VQEとは何か？

VQEは：

ある行列の

最小固有値

を求めるアルゴリズムです。

つまり：

エネルギー最小状態

を探索します。

---

# なぜ最小固有値が重要なのか？

最小固有値は：

次の問題に現れます：

* 分子の基底エネルギー
* 材料科学
* 化学反応シミュレーション
* 最適化問題

つまり：

現実世界の問題に直結します。

---

# なぜVQEがNISQ向きなのか？

従来の量子アルゴリズムは：

深い回路

を必要としました。

しかし：

現在の量子コンピュータは

ノイズが多い

という制約があります。

VQEは：

浅い回路

＋

古典最適化

で動きます。

これが：

NISQアルゴリズム

の特徴です。

---

# VQEの基本アイデア

VQEは：

次の流れで動きます：

1. パラメータ付き回路を作る
2. 状態を生成する
3. エネルギーを測定する
4. 古典コンピュータで最適化する
5. 繰り返す

つまり：

量子＋古典

のハイブリッド計算です。

---

# 最も単純な例を考える

今回は：

次のハミルトニアンを最小化します：

H = Z

つまり：

Zの最小固有値

を求めます。

---

# なぜこれでよいのか？

Zの固有値は：

+1
-1

です。

つまり：

最小値は

-1

になります。

これを：

VQEで求めます。

---

# パラメータ付き回路を作る

まず：

回転ゲートを使います。

コード：

```python id="q6c2fv"
from qiskit import QuantumCircuit
from qiskit.quantum_info import Statevector
import numpy as np

def circuit(theta):
    qc = QuantumCircuit(1)
    qc.ry(theta, 0)
    return qc
```

この回路は：

θによって状態が変わります。

---

# 期待値を計算する

次に：

Zの期待値を計算します。

コード：

```python id="o1k39y"
def expectation(theta):
    state = Statevector.from_instruction(circuit(theta))
    return state.expectation_value([[1, 0], [0, -1]]).real
```

これが：

エネルギーになります。

---

# θを変えてみる

試しに：

いくつかの値を計算します。

```python id="4jxj2j"
for theta in np.linspace(0, 3.14, 5):
    print(theta, expectation(theta))
```

結果：

θによって

エネルギーが変わります。

---

# 最小値を探索する

次に：

最小値を探します。

コード：

```python id="7n3px8"
from scipy.optimize import minimize

result = minimize(expectation, x0=0)

print(result)
```

これで：

最小値が求まります。

---

# なぜ最小値が求まるのか？

回転ゲート：

RY(θ)

は：

状態を回転させます。

つまり：

θを変えることで

状態空間を探索できます。

その中で：

最小エネルギー状態

を探します。

---

# VQEの本質

VQEは：

次の式を最小化しています：

⟨ψ(θ)|H|ψ(θ)⟩

つまり：

期待値の最小化

です。

ここで：

ψ(θ)

は：

回路で生成された状態

です。

---

# 実際のVQEはもっと複雑

現実の問題では：

ハミルトニアンは：

H = a₁Z + a₂X + a₃ZZ + …

のようになります。

つまり：

複数の項の和

になります。

しかし：

基本構造は同じです。

---

# VQEが重要な理由

VQEは：

現在の量子コンピュータで

最も実用的なアルゴリズム

です。

応用例：

* 分子シミュレーション
* 材料設計
* 量子化学
* 最適化問題

---

# 古典計算との役割分担

VQEでは：

量子コンピュータ：

状態生成

古典コンピュータ：

パラメータ更新

を担当します。

つまり：

ハイブリッドアルゴリズム

です。

---

# なぜハイブリッドが重要なのか？

現在の量子コンピュータは：

単独では弱い

からです。

しかし：

古典計算と組み合わせると

強力になります。

これが：

NISQ時代の戦略

です。
