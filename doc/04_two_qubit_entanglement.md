# 2量子ビットとは何か？

1量子ビットは：

|ψ⟩ = a|0⟩ + b|1⟩

でした。

では：

2量子ビットはどうなるでしょうか？

答えは：

|ψ⟩ = a|00⟩ + b|01⟩ + c|10⟩ + d|11⟩

になります。

つまり：

状態が4つになります。

| 状態 |     |
| -- | --- |
|    | 00⟩ |
|    | 01⟩ |
|    | 10⟩ |
|    | 11⟩ |

です。

---

# 状態数は指数的に増える

量子ビットが増えると：

状態数はこうなります：

| 量子ビット数 | 状態数   |
| ------ | ----- |
| 1      | 2     |
| 2      | 4     |
| 3      | 8     |
| 10     | 1024  |
| 20     | 約100万 |

つまり：

量子ビットが増えると

表現できる状態空間が爆発的に増えます。

これが：

量子計算の強さの源です。

---

# 2量子ビット回路を作る

まず：

2量子ビット回路を書いてみます。

コード：

```python
from qiskit import QuantumCircuit
from qiskit.quantum_info import Statevector

qc = QuantumCircuit(2)

state = Statevector.from_instruction(qc)

print(state)
```

結果：

[1.+0.j, 0.+0.j, 0.+0.j, 0.+0.j]

つまり：

|00⟩

です。

これは：

初期状態です。

---

# 1つ目の量子ビットだけ操作する

次に：

最初の量子ビットにHadamardを適用します。

```python
from qiskit import QuantumCircuit
from qiskit.quantum_info import Statevector

qc = QuantumCircuit(2)

qc.h(0)

state = Statevector.from_instruction(qc)

print(state)
```

結果：

[0.707, 0, 0.707, 0]

つまり：

0.707|00⟩ + 0.707|10⟩

です。

ここで重要なのは：

1つの量子ビットを操作すると

全体の状態が変わる

という点です。

---

# CNOTゲートとは何か？

ここで：

最も重要な2量子ビットゲート：

CNOT

を導入します。

CNOTとは：

Control NOT

です。

意味：

制御ビットが1のときだけ
ターゲットビットを反転する

という操作です。

表で書くと：

| 入力 | 出力 |
| -- | -- |
| 00 | 00 |
| 01 | 01 |
| 10 | 11 |
| 11 | 10 |

です。

---

# CNOTを実行する

では：

実際に回路を書いてみます。

```python
from qiskit import QuantumCircuit
from qiskit.quantum_info import Statevector

qc = QuantumCircuit(2)

qc.h(0)

qc.cx(0, 1)

state = Statevector.from_instruction(qc)

print(state)
```

結果：

[0.707, 0, 0, 0.707]

つまり：

0.707|00⟩ + 0.707|11⟩

になりました。

これは：

ベル状態（Bell state）

と呼ばれます。

---

# ベル状態とは何か？

ベル状態は：

最も基本的なエンタングル状態です。

例：

|Φ⁺⟩ = 0.707|00⟩ + 0.707|11⟩

特徴：

2つの量子ビットが

完全に結びついています。

---

# エンタングルメントとは何か？

通常の状態：

0.707|00⟩ + 0.707|10⟩

これは：

次のように分解できます：

(0.707|0⟩ + 0.707|1⟩) ⊗ |0⟩

つまり：

独立した状態です。

しかし：

ベル状態

0.707|00⟩ + 0.707|11⟩

は：

分解できません。

つまり：

2つの量子ビットが

独立ではありません。

これが：

エンタングルメント

です。

---

# 測定すると何が起きるか？

ベル状態を測定すると：

結果は：

|00⟩ または |11⟩

になります。

しかも：

必ず一致します。

つまり：

片方が0なら
もう片方も0

片方が1なら
もう片方も1

になります。

これは：

古典的な相関とは異なる

量子的な相関です。

---

# 回路として理解する

ベル状態は：

次の2ステップで作れます：

1. Hadamard
2. CNOT

つまり：

H → CNOT

だけです。

コード：

```python
qc = QuantumCircuit(2)

qc.h(0)
qc.cx(0, 1)
```

これが：

量子もつれを作る最小回路

です。

---

# なぜエンタングルメントが重要なのか？

エンタングルメントは：

量子アルゴリズムの核心です。

例えば：

* Grover
* Shor
* QFT
* VQE
* QAOA

すべて：

エンタングルメントを使います。

つまり：

エンタングルメントなしでは

量子計算は成立しません。

---

# 実験：測定してみる

測定を追加してみます。

```python
from qiskit import QuantumCircuit
from qiskit_aer import AerSimulator
from qiskit.compiler import transpile

qc = QuantumCircuit(2, 2)

qc.h(0)
qc.cx(0, 1)

qc.measure([0, 1], [0, 1])

simulator = AerSimulator()

compiled = transpile(qc, simulator)

job = simulator.run(compiled, shots=1000)

result = job.result()

print(result.get_counts())
```

結果例：

{'00': 503, '11': 497}

つまり：

00 と 11 しか出ません。

これが：

エンタングルメントの証拠です。

---

# エンタングルメントはなぜ量子らしいのか？

古典ビットなら：

2つのビットは

常に独立です。

しかし：

量子ビットは：

独立でない状態を作れます。

つまり：

状態空間そのものが共有されます。

これが：

量子計算の本質です。
