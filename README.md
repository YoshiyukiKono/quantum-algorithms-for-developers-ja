## 本リポジトリについて

本リポジトリは、

**量子コンピューティングの基礎を理解した開発者が**
**実際に量子アルゴリズムを実装できるようになる**

ことを目的とした実践シリーズです。

対象読者：

- Python が書ける開発者
- 量子ビット・干渉・エンタングルメントを理解している人
- Qiskit をこれから使い始めたい人
- NISQアルゴリズムを実装したい人

このシリーズでは：

- 回路を書く
- アルゴリズムを実装する
- ノイズを扱う
- 実機で動かす

ところまで到達します 🚀
